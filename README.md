# Sales Data Cleaning and Analysis Project

## Project Overview
This project focuses on cleaning and analyzing a messy sales dataset using Python.

## Tools Used
- Python (pandas, numpy)
- Excel

## Data Cleaning Steps
- Handled missing values (NaN)
- Removed duplicate records
- Fixed incorrect data types
- Converted Date column to datetime
- Created new features (Month, Year)

## Analysis Performed
- Sales by Product
- Sales by Region
- Monthly Sales Trends
- Top Orders by Revenue

## Key Insights
- Laptop generates highest revenue
- North region shows strong performance
- Seasonal trends observed in sales

## 🚀 Outcome
Created a clean, structured dataset ready for business analysis and visualization.
